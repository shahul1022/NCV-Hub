Share with Your Team — Free
This tool is a single HTML file — host it anywhere, free forever
✅ 100% Free · Works in any browser · No account needed for viewers
Host the file once, share the URL with your team. Everyone opens it directly in their browser — nothing to install.
Option A — Netlify Drop (fastest · 30 seconds · free)

Recommended for non-technical teams

1. Download the file above
2. Go to app.netlify.com/drop
3. Rename your file to index.html, then drag and drop it onto the page
4. You instantly get a URL like https://your-name.netlify.app
5. Share that link in your team WhatsApp/email — done!

Option B — GitHub Pages (free forever, more control)

Best for technical teams

1. Create a free account at github.com
2. Create a new public repository (e.g. ncv-hub)
3. Upload file renamed as index.html
4. Settings → Pages → Branch: main → Save
5. URL: https://yourname.github.io/ncv-hub

Option C — SharePoint or Google Drive

Already have company tools? Use them

Upload the HTML file to your company SharePoint or Google Drive and share the view link. Team members click to open it directly in Chrome/Edge. No hosting setup needed.

⚠️ About data
Each person's browser stores their own data. For shared live data across the whole team (everyone sees the same projects), ask your Claude to connect a free Supabase backend — that upgrade takes about 30 minutes.
